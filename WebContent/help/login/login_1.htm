 로그인 비밀번호 변경 방법
아이디와 비밀번호는 그룹웨어 사용 시 자신을 확인하는 고유한 이름과 암호이므로 보안에 각별한 주의가 필요합니다.

(1) 로그인 화면의 비밀번호 변경 버튼 클릭
(2) ID : 자신의 사번 입력
     현재 비밀번호 : 현재 비밀번호 입력
     새 비밀번호 : 변경하고자 하는 비밀번호 입력
     비밀번호 확인 : 변경하고자 하는 비밀번호 재 입력(확인용)
